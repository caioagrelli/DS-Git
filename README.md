# Portfólio Git — Caio Agrelli

Repositório pessoal desenvolvido para o exercício **EI1** da disciplina **CIN0136 — Desenvolvimento de Software** (CIn-UFPE, 2026.1).

O objetivo é demonstrar o uso profissional de Git: commits semânticos, branches, Pull Requests e resolução de conflitos.

---

## Estrutura

```
seu-portfolio-git/
├── README.md              ← este arquivo
├── REFLEXAO.md            ← reflexão pessoal sobre o processo
├── exercicios/
│   ├── ex01-commits/      ← documentação sobre commits semânticos
│   ├── ex02-branches/     ← documentação sobre branches e merges
│   └── ex03-conflito/     ← registro da resolução de conflito
└── projeto/
    └── hello-word.html    ← Jogo da Forca (mini-projeto)
```

---

## Exercícios

### [ex01-commits](./exercicios/ex01-commits/README.md)
Prática com commits semânticos seguindo o padrão Conventional Commits. Inclui tabela com todos os commits realizados e seus tipos.

### [ex02-branches](./exercicios/ex02-branches/README.md)
Prática com criação e gerenciamento de branches. Documenta todas as branches criadas, o fluxo de trabalho e o que foi aprendido.

### [ex03-conflito](./exercicios/ex03-conflito/REGISTRO.md)
Evidência e documentação da criação e resolução de um merge conflict real.

## Mini-projeto

O **Jogo da Forca** (`projeto/hello-word.html`) é uma página web single-file em HTML/CSS/JS com:

- Interface responsiva com teclado virtual
- Tema claro/escuro com toggle
- Animações e feedback visual
- Frase surpresa relacionada à disciplina

Abra o arquivo diretamente no navegador para jogar.
